# Erin

Define your browser newtab page by ❤️.

用极简地风格重新定义你的浏览器新标签页

更多内容，请访问：[Erin - 极简的 chrome 新标签页插件](https://erin-homepage.vercel.app/)



## Preview

![](https://i.imgur.com/MF3jWNF.jpeg)

## Feature

- [x] 壁纸功能

- [x] 稍后阅读列表

- [x] 壁纸下支持书签

- [x] tab 树管理，支持 newtab、content、popup 三个页面显示

- [x] 搜索引擎切换、搜索建议、搜索框

- [x] 支持 markdown 的 note 面板

- 菜单增强
  - [x] 图片下载
  - [x] copy as markdown
  - [x] 画中画
